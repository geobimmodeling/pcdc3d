# c{reate}  < [range](range/) >
pcdc b{olt}-e{lement} <span style='color: red;'>c{reate}</span>
> **描述：**给定起始点坐标创建支护单元，并且创建的锚杆单元在给定有限元单元的范围内， 不在范围内的长度部分被自动裁剪删除。

> 
> **值：**
> 
> v1（[vector](数据类型/vector/)，支护锚杆的起点坐标）
> 
> v2（[vector](数据类型/vector/)，支护锚杆的终点坐标）
