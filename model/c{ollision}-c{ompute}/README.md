# c{ollision}-c{ompute}
pcdc model <span style='color: red;'>c{ollision}-c{ompute}</span>
> **描述：**设置接触判断力的算法
> 
> **默认值：**accurate

**子关键词：**[iterate](model/c{ollision}-c{ompute}/iterate/)，[accurate](model/c{ollision}-c{ompute}/accurate/)，


**举例：**
```
pcdc model c-c iterate
#该命令设置接触判断为SAP算法

```
